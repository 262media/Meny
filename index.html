Meny

A UI experiment with a 3D fold-in menu. Move your mouse to the left edge of this page — or swipe to the right if you're on a touch device — to expand the menu.

This was created using CSS 3D transforms. JavaScript is used to track mouse/touch movement. The source code is available at github.com/hakimel/meny.
